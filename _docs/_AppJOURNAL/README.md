# App Journal

The app journal is the append-only execution ledger and continuation surface for UsefulHELPER.

## Canonical Surfaces

- SQLite database: `_docs/_journalDB/app_journal.sqlite3`
- entry mirrors: `_docs/_AppJOURNAL/entries/`
- current tasklist mirror: `_docs/_AppJOURNAL/CURRENT_TASKLIST.md`
- backlog mirror: `_docs/_AppJOURNAL/BACKLOG.md`
